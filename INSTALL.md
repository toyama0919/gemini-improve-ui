# Gemini Chat UI Improvements - Installation Guide

## How to Load the Extension

1. **Open Chrome**

2. **Navigate to the Extensions page**
   - Type `chrome://extensions/` in the address bar and press Enter

3. **Enable Developer Mode**
   - Toggle "Developer mode" ON in the top right corner

4. **Load the extension**
   - Click the "Load unpacked" button
   - Select the `gemini-improve-ui` directory

5. **Verify**
   - Confirm that "Gemini Chat UI Improvements" appears in the extension list
   - Make sure the extension is enabled

6. **Start Using**
   - Navigate to `https://gemini.google.com/app`
   - Use the keyboard shortcuts

## Troubleshooting

### If the extension doesn't work

1. Click the refresh button on the extension page (`chrome://extensions/`)
2. Reload the Gemini page (F5 or Cmd+R)
3. Check for errors in the DevTools Console tab (F12)
