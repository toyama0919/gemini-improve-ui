# Supercharge Your Gemini Chat Experience with Keyboard Shortcuts

If you're like me and spend a significant amount of time chatting with Google Gemini, you've probably noticed that constantly reaching for your mouse can slow you down. That's why I built **Gemini Chat UI Improvements** - a Chrome extension that brings vim-like keyboard navigation to Gemini Chat.

## The Problem

Google Gemini's chat interface is great, but it's heavily mouse-dependent. Simple tasks like:
- Switching between conversations
- Navigating search results
- Copying code snippets
- Creating new chats

...all require you to move your hands away from the keyboard. For power users who prefer keyboard-driven workflows, this context switching adds up to significant time waste.

### Key Design Decisions

**1. Modular Architecture**
Each feature is encapsulated in its own module, making the codebase maintainable and easy to extend.

**2. Non-Invasive DOM Manipulation**
The extension observes DOM changes and adds functionality without modifying Gemini's core behavior. This ensures compatibility even when Google updates their interface.

**3. User-First Configuration**
All shortcuts are stored in Chrome's sync storage, allowing users to maintain their preferences across devices.

**4. State Management**
The history selection mode uses a simple state machine to manage different interaction modes without conflicting with Gemini's native functionality.

## Installation

Since this is a developer-focused tool, installation is straightforward:

```bash
# Clone the repository
git clone https://github.com/toyama0919/gemini-improve-ui.git
cd gemini-improve-ui
```

Then load it in Chrome:
1. Navigate to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `gemini-improve-ui` directory

Visit `https://gemini.google.com/` and start using the shortcuts immediately!
